# Higher Lower Game 

A command-line implementation of the popular **Higher Lower** game built with Python.

# What the game does:
Players are presented with popular instagram accounts and must guess which one has **more followers**. Each correct guess increases the player's score, while an incorrect guess ends the game.

The project uses a dataset of real-world accounts and demonstrates how to work with lists, dictionaries, functions, and game logic in Python.

## Features in this project:

* Randomly selects two accounts for comparison
* Displays each account's name, description, and country
* Tracks the player's score throughout the game
* Correct guesses allow the player to continue playing
* The winning account carries over to the next round
* Simple command-line interface with ASCII art

## Concepts Practiced

* Functions and modular programming
* Lists and dictionaries
* Accessing nested data structures
* Conditional statements (`if`, `else`)
* Loops (`while`)
* Boolean logic
* Random selection using Python's `random` module
* Score tracking
* User input handling
* Game state management
